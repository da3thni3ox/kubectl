
### Переменные окружения  
```
KUBE_CA_CERT - корневой сертификат (удостоверяющий)
KUBE_ENDPOINT - адрес кластера k8s 
K8S_CLUSTER_NAME - название кластера (hostname)
K8S_DEPLOY_USER - пользователь кластера
KUBE_ADMIN_CERT - сертифкат пользователя
KUBE_ADMIN_KEY - ключ сертификата пользователя
```
